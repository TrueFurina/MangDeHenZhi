> 编制：UX Researcher ｜ 日期：2026-07-17 ｜ 配套：`mangdehenzhi-user-research-plan.md`
> 目标：识别演示雷区与体验断点（H-C1 模拟上链理解、H-D1 写死路径、H4 个性化、H5/H7 实训真实感与眩晕、B5 报名缺失）。

### 2.2 招募渠道
高校社群 / 求职社区（牛客、脉脉小组）/ 职场微信群 / 残障人士友好社区（U5）。

### 2.3 筛选问卷（发邀前 5 题）
1. 近 3 个月是否用过职业技能/求职类平台？（是/否——优先"是"）
2. 当前身份：大学生 / 应届生 / 转行(25–35) / 在职 / 其他
3. 自评数字素养：低 / 中 / 高
4. 是否有求职或转行计划（6 个月内）？（是/否）
5. 是否使用屏幕阅读器（仅 U5 渠道问）

### 2.4 邀请文案（可直接发）
> 【芒得很职 · 用户体验邀请】我们在做一款"AI 测评 + 虚拟实训 + 技能证书"的职培平台，想请你用 45 分钟体验并提意见。全程录像（可匿名），赠送 ¥50 京东卡。地点：线上会议 / 线下均可。感兴趣私信我～

### 2.5 偏差防止
- 排除内部员工/亲友；避免 5 人全为学生（U2/U3 必含）；
- 不提前告知"这是演示版/模拟"，让其自然反应（H-C1 对照在问卷侧做，不在引导中提示）。

## 4. 引导稿（45 分钟 / 人）

### 开场（5 分钟）
- 欢迎、说明目的（"我们测产品不是测你，你的任何反应都很有价值"）；
- 签署**知情同意书**；
- 讲解**出声思考（think-aloud）**："做的时候把脑子里想的都说出来"；
- 采集背景（身份/用过哪些职培产品/近期求职或学习目标）。

### 基线（5 分钟）
- 你平时怎么学新技能？用过哪些平台？
- 对"AI 给我打分""在虚拟环境练面试""证书上链"有听过吗？第一反应？

### 任务（25 分钟，见任务卡）
按顺序走 T1→T5（U5 仅做 T1/T3/T5 的键盘+读屏版）。

### 收尾（10 分钟）
- 整体印象、最惊喜 / 最困惑的一点；
- 如果这是收费产品，你愿意为哪部分付钱？（开放）；
- 填写 **SUS + 补充量表**（第 6 节）。

## 5. 任务卡（成功标准 / 指标 / 观察点）

**T1｜注册登录 → 完成 12 题测评 → 解读雷达图**
- 成功：看到 AI 报告 + 能力雷达图 + 推荐。
- 指标：完成率、时长、错误数（如进度卡住、提交失败）。
- 观察：进度条/圆点导航是否清晰；低素养用户（U4）是否在长流程中放弃（验证 H-E1）。

**T2｜查看 AI 推荐课程 → 打开课程详情 → 尝试报名**
- 成功：进入课程详情页。
- 指标：是否尝试"报名"、受阻反应。
- 观察：① **B5 报名缺失**（仅 ElMessage 提示）必现，记录用户困惑；② **H-D1 写死路径**是否被感知为"不准/无关"；③ H4 个性化缺失的主观反馈。

**T3｜进入"我的证书"，解释"模拟上链"含义**
- 成功：能用自己的话解释这张证书目前是什么状态。
- 指标：理解度（正确/部分/误解）；对"模拟"标注的信任反应。
- 观察：**H-C1** —— 标注是否被理解？是否降低价值感知？有无"这是骗人"的负面解读（合规风险信号）。

**T4｜进入元宇宙静态场景，旋转视角，描述所见**
- 成功：完成 OrbitControls 旋转，能描述场景元素。
- 指标：眩晕/卡顿主观报告（1–5）、可理解性。
- 观察：**H5**（真实感/说教感）、**H7**（眩晕率~15%，优化后3%）——记录任何不适；确认"减少动效"开关需求。

**T5｜无障碍走查（U5）**
- 仅键盘 / 屏幕阅读器完成 T1 的"登录→看到报告"。
- 观察：焦点顺序、读屏可读、对比度、3D 场景是否完全不可达（需替代文本/说明）。

## 6. 问卷（测试后填写）

### 6.1 SUS（系统可用性量表，标准 10 题，1=强烈不同意→5=强烈同意）
1. 我认为我会经常使用这个系统。
2. 我发现这个系统不必要地复杂。
3. 我认为这个系统很容易使用。
4. 我需要技术支持才能使用这个系统。
5. 我发现各个功能很好地集成在一起。
6. 我认为系统里有很多不一致的地方。
7. 我想象大多数人会很快学会使用这个系统。
8. 我发现这个系统使用起来很麻烦。
9. 我对使用这个系统很有信心。
10. 使用前我需要学习很多东西。
> 计分：奇数题 (得分-1)，偶数题 (5-得分)，求和×2.5 → 0–100。基准 ≥68 为行业平均。

### 6.2 补充量表（5 点李克特）
- **模拟上链理解度**：我能清楚理解"模拟上链"是什么意思。（验证 H-C1）
- **信任反应**：标注"模拟"让我对证书的信任（1=大幅下降 → 5=无影响）。
- **3D 眩晕/不适**：体验 3D 场景时我感到（1=严重不适 → 5=完全无感）。（验证 H7）
- **实训真实感**：3D 场景让我感觉"在练习"（1=纯展示 → 5=身临其境）。（验证 H5）
- **路径相关性**：推荐课程/路径与我的情况相关（1=完全无关 → 5=很准）。（验证 H-D1/H4）

## 8. 分析模板（测完即填）

1. **问题清单**：每条问题记「出现频次（n/5）+ 严重程度（1–5）+ 关联假设」；
2. **痛点矩阵**：严重度 × 普遍度（频次/5）定位优先；
3. **假设判定**：H-C1 / H-D1 / H4 / H5 / H7 / B5 / H-E1 各给「支持 / 削弱 / 待定量验证」；
